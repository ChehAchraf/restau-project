# About the Project

The project is a web template for a restaurant specializing in selling desserts and baked goods. It showcases products on different pages while ensuring communication with customers and users.

## Site Structure

The website consists of 4 pages as follows:

1. **Home Page**: This page presents the main idea of the restaurant.
2. **Best Products Page**: Highlights the top products offered.
3. **Contact Us Page**: Provides a way for customers to get in touch.
4. **Menu Page**: Displays the menu provided by the restaurant.
5. **About Us Page**: Offers a brief overview or story about the restaurant.
